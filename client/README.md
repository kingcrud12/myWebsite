# Dossier Client

Ce dossier contient tous les fichiers frontend du site.

## Fichiers

- **index.html** : Page principale du site
- **styles.css** : Feuille de style CSS
- **script.js** : Scripts JavaScript pour les interactions

## Structure

Les fichiers sont organisés de manière à être servis directement par le serveur web.

## Chemins relatifs

- Les fichiers CSS et JS sont référencés de manière relative dans `index.html`
- Le script JavaScript pointe vers `../server/send-email.php` pour l'envoi d'emails
